whatsApi.getProfilePicture(target, small, callback)

Send a request for the profile picture for the specified account

When received from server a profile.picture event is fired When profile picture can not be retrieved an error 404 item-not-found is returned

Kind: instance method of WhatsApi

Param Type Description
target String Phonenumber of the account to request profile picture from
small Boolean true for thumbnail, false for full size profile picture
callback ProfilePictureCallback

Example 

// Request full size profile picture from 49xxxxxxxx
wa.getProfilePicture('49xxxxxxxx', false, function(res) {
  fs.writeFile('whatsapi/media/profilepic-'+res.from+(res.isPreview?'-preview':'-full')+'.jpg', res.pictureData); 
});
